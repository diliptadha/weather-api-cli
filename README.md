# Weather API CLI

### This is a command line interface for weather fetching.

## clone the repo

```bash
git clone https://github.com/diliptadha/weather-api-cli.git
```

## install dependencies

```bashe
npm install
npm link
```

## run

`node index.js`
or
`weather`

## API for fetching weather data

https://open-meteo.com/en/docs
